令和七年四月一日（以下「適用日」という。）の前日において任期付職員法第七条第三項の規定による俸給月額を受けていた職員の適用日における俸給月額は、その者の適用日の前日における俸給月額に対応する次の表の新俸給月額欄に定める俸給月額とする。適用日の前日における俸給月額新俸給月額円円９８８，０００１，０２１，０００１，１１２，０００１，１４９，０００１，１９１，０００１，２２４，０００

| 適用日の前日における俸給月額 | 新俸給月額 |
| --- | --- |
| 円 | 円 |
| ９８８，０００ | １，０２１，０００ |
| １，１１２，０００ | １，１４９，０００ |
| １，１９１，０００ | １，２２４，０００ |

## 附則

この規則は、公布の日から施行する。
