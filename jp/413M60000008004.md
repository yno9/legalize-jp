**第一条（本省及び消防庁の定員）**　総務省の本省及び消防庁の定員は、次の表のとおりとする。区分定員備考本省四、五七六人　消防庁一七四人　合計四、七五〇人　

| 区分 | 定員 | 備考 |
| --- | --- | --- |
| 本省 | 四、五七六人 |  |
| 消防庁 | 一七四人 |  |
| 合計 | 四、七五〇人 |  |

**第二条（本省及び消防庁の各内部部局、各審議会等、各施設等機関、各特別の機関及び各地方支分部局別の定員）**　本省及び消防庁の各内部部局、各審議会等、各施設等機関、各特別の機関及び各地方支分部局別の定員は、前条に定める本省又は消防庁の定員の範囲内において、総務大臣が別に定める。
