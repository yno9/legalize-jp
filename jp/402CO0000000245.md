**第一条（素材等）**　天皇陛下御即位記念のための十万円の貨幣の発行に関する法律第一条の規定により発行する貨幣の素材、品位、量目及び形式を次のように定める。素材金品位純金量目三十グラム形式直径三十三ミリメートル

| 素材 | 金 |  |  |
| --- | --- | --- | --- |
| 品位 | 純金 |  |  |
| 量目 | 三十グラム |  |  |
| 形式 |  | 直径 | 三十三ミリメートル |
|  |  |  |  |

**第二条（発行枚数）**　前条に規定する貨幣の発行枚数は、二百万枚とする。

**第三条（販売価格）**　特殊な技術を用いて製造し表面に光沢を持たせた第一条に規定する貨幣で一枚を容器に入れたものの販売価格は、十一万三千三百円とする。

## 附則

この政令は、公布の日から施行する。
